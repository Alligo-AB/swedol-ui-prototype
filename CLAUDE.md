# Project Context

This repository is a lightweight UI prototype for sharing design concepts with the team.

## Goal
Keep the project simple, static, and easy to preview in Vercel.

## Rules
- Do not add frameworks unless explicitly requested.
- Keep everything as plain HTML + CSS unless there is a strong reason to change.
- Prioritize readability and simple structure.
- Do not rename files unless necessary.
- Keep the site deployable as a static site on Vercel.

## File Structure
- `index.html` = main page
- `styles.css` = all styling
- `README.md` = setup and workflow notes

## Workflow
- Make small changes
- Commit with clear messages
- Push to `main`
- Vercel will generate a live deployment preview

## Design Notes
- This repo is for prototype/handoff work, not production Magento code.
- The live Vercel URL is the share link for reviews.
